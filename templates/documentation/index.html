Technology Stack

Frontend

Languages:

  • HTML
  • CSS
  • Javascript

Frameworks:

  • Bootstrap 5.2.1
  • jQuery 3.6.1 (CDN)
  • Axios 1.2.2 (CDN)
  • Popper 2.11.6 (CDN)

Miscellaneous

  • Font Awesome


Backend

Languages:

  • Python 3.8 (As of 1/6/23 - Railway uses Nixpacks which defaults to 3.8)

Frameworks:

  • Flask 2.2.2
    • Flask-SQLAlchemy 2.5.1
    • Flask-Bcrypt 1.0.1
    • Flask-WTF 1.0.1

Database:

  • PostgreSQL 13 (As of 1/6/23 - Railway uses PSQL 13 Docker image

API:

  • SendGrid (Email Service)

Storage

  • AWS S3 (File Storage)


Host

  • Railway
