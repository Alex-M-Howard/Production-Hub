Offline Program Loading

Purpose

Purpose: Load Trumpf laser programs to the machine without a network connection
Scope: Applies to the Trumpf fiber laser
Responsibility: Manufacturing Engineering: Utilize this strategy when a network issue arises to continue laser production
Procedure:
Change Transfer Directory 1. Change the transfer directory to an offline directory in the Machine Tab in TruTops
2. Choose program you wish to load in the nesting tab, click the unrelease button, and then click it again to re-release the program in the new transfer directory
3. Copy the programs from the transfer directory to a flash drive to load to the machine controller
4. Document which programs were released in the new transfer directory
5. Once the network error has been fixed: 5.1. Change the Transfer Directory back to the original (\\PRIZ-MERRYMECH\TruLaser2030) 5.2. Repeat step 5.2 for all documented programs from step 4 5.3. Delete all programs from flash drive to prevent issues with rev changes in the future
