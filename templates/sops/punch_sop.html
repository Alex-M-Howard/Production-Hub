Punch

Purpose

Purpose: Document the steps needed to program a part using MerryMech SMP software
Scope: Applies to new AND revised parts to be run on the turret punch press
Responsibility: Engineering: Notify programming/fabrictation of the new or revised part and include a PDF drawing and DXF file in the appropriate folders

Programmer: Revise/release the part for fabrictation to run on the turret punch press
Procedure:

Create Single Part (with parts directory open): 1. Check DXF for programmability (should be engineering)
2. Find the appropriate folder for the part in merry mech (dual fuel, 600 series, wall oven, etc.)
3. Click new 3.1. Create new part selected 3.2. Override if exists checked 3.3. Import from cad file 3.4. Search "..." 3.5. Find dxf file (pending, released, etc.), click enter 3.6. Remove rev from part number, click ok 4. Part details box 4.1. Enter rev level number 4.2. Enter part description 4.3. Enter your name and date 4.4. Select part grain ng = no grain gx = right left grain gy = top to bottom grain 4.5. Optional note - enter ECO number, rev number, replaced by, etc. 4.6. Material tab 4.6.1. Select material, and generic 48x120 sheet size 4.7. Method info tab 4.7.1. Select machine 6765 = Murata Punch 1798 = Finn Reselect sheet size and material in this tab 4.8. Click apply, and OK 5. If internal edge not closed, box shows up 5.1. To remove bend lines, click exit to geometry 5.1.1. Modify, delete, single 5.1.2. Click all, right click to confirm 5.1.3. Left click to delete bend lines 5.2. To leave the scribe geometry if it has scribe info, scribe all after delete bend lines 6. Click NC
7. Tool load 7.1. Graphic tool load manager 7.2. Confirm all tools are loaded and in proper stations 8. Lay hits appropriately using proper programming practice 8.1. Typical tabbing at 0.01" to 0.02" 8.2. File exit, save
Create nest (with directory open):   1. Click new   
  2. Create new nest button selected   
  3. Create nest number (part number for single part sheets with N in the front, hoods are the weld assembly number), click OK   
  4. Fill out rev and description   
  5. Enter your name and date   
  6. Enter grain as GX unless it is a blank size   
  7. Fill out optional note (if it will be put on the laser enter "laser") leave blank if not on laser   
  8. Material tab   
8.1. Choose sheet size and material, this can be modified later"   9. Method tab   
9.1. Choose machine number" 6765 = Murata Punch 1798 = Finn 10. Apply, OK
11. Add part or parts 11.1. Find parts in part directory 11.2. Click on part, right click on nest buffer for the proper machine number 11.3. Add to nest, drag and place appropriately 11.4. Spacing from clamps at bottom of sheet 11.4.1. Y-location min 3.645" for Murata (6765) 11.4.2. Y-location min 3.0" for Finn (1798) 12. Exit and save do not process if only on laser
13. Exit save, process nest
14. Kit page 14.1. Remove duplicates 14.2. Modify, remove duplicates 14.2.1. NOTE YOU MUST MANUALLY REMOVE DUPLICATES IF YOU HAVE SCRIBE 14.2.2. Modified, delete, delete to manually remove single hits 14.3. Concatenate, yes 14.4. Right click, keep existing tabs, yes 14.5. If bottom Y spacing is less than min listed above, you will need to simplify hits on bottom row to clear clamps 14.5.1. Modify, delete, simplify 14.6. Process into multiples button 14.6.1. Right click, right click, right click 15. Multiples page window 15.1. Verify sheet size is correct 15.2. Adjust trim spacing to min Y amount listed above 15.3. Set appropriate borders, OK 16. Nest page 16.1. Check repo to see what hits need to be held, add, repo, delete repo, last repo 16.2. Hold large hits 16.2.1. Modify, hold hits, single hits 16.2.2. Right click to finish 16.3. Left click sequencing button 16.3.1. Reorder station appropriately smallest to largest, scribe-rounds-hex-obrounds-radii-square-rec-form 16.3.2. Sequence hits appropriately 16.4. Exit, add, repo, shift 16.4.1. Repeat sequence step 16.4.2. Final sequence end at clamps to keep structure 16.5. Clamp clear if necessary (less than min Y distance) 16.6. Write code 16.6.1. Program number is last 4 of N number, if underscore x, still use last 4 numerical digits 16.6.2. Enter, add material codes see chart, enter 16.6.3. "1" yes to add unload code, enter 17. Transfer code to machine file 17.1. Files save locations by machine 17.1.1. (6765) \\PRIZ-MERRYMECH\merrymech_files\6765 17.1.2. (1798) \\PRIZ-MERRYMECH\merrymech_files\1798 17.2. For machine 6765 17.2.1. Move .nc1 (code) files to "NCdata1 (prizer-campath)(V)" for 6765, place in respective folder (electric walloven, hodds, etc.) 17.2.2. Move .op1 files from op folder in location above to "NCdata1 (prizer-campath)(V)" "z folder" 17.3. For machine 1798 17.3.1. Move files to "NCPROG" folder in \\prizerfinnpower computer
